# Scratchpad

## Overview
Temporary workspace for notes, drafts, and experiments that should not yet live in a package-specific directory.

## Current Contents
- `scratchpad/README.md`

## Suggested Usage
- Keep short-lived notes and test snippets here.
- Move durable content into the relevant package README or script docs.
- Periodically clean stale files.

## Naming
Suggested conventions:
- `YYYY-MM-DD-topic.md`
- `tool-name-experiment.md`
- `TODO-topic.md`
