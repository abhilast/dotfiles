Fix any errors found:
- Template syntax errors: fix the .tmpl file
- Source/target drift: run `chezmoi add ~/.the-file` to pull the live version into source
- Missing files: verify the copy commands in Plans 02/03 ran correctly

After fixing, re-run until `chezmoi apply --dry-run` produces no errors.
