# Btop Configuration

## Overview
Btop config is tracked at:
- `btop/.config/btop/btop.conf`

## Install
```bash
cd ~/dotfiles
stow btop
```

## Current Config Highlights
- `color_theme = "TTY"`
- `vim_keys = false`
- `shown_boxes = "cpu net proc mem"`
- `update_ms = 100`

## Usage
```bash
btop
```

Use `h` inside btop for the up-to-date key reference for your installed version.
